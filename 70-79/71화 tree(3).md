### 이진 탐색 트리

- 자료구조 입력, 탐색 비교

| 자료구조 | 입력     | 탐색     |
| -------- | -------- | -------- |
| vector   | O(1)     | O(N)     |
| list     | O(1)     | O(N)     |
| BST      | O(log N) | O(log N) |

- 트리의 순회

| 종류                  | 순서                 |
| --------------------- | -------------------- |
| 전위 순회 (preorder)  | 부모 - 왼쪽 - 오른쪽 |
| 중위 순회 (inorder)   | 왼쪽 - 부모 - 오른쪽 |
| 후위 순회 (postorder) | 왼쪽 - 오른쪽 - 부모 |

- 이진 탐색 트리를 중위 순회 하면 정렬된 순서대로 순회할 수 있다.
- 단, 입력이 반드시 밸런스있게 들어온다는 보장이 없기에 'self balanced BST'를 구현한다.
- 대표적으로 AVL, 레드-블랙 트리가 있다.
