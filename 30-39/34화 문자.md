### 문자

- 메모리 영역
  - 1. 스택
  - 2. 데이터
  - 3. 힙
  - 4. ROM(코드)

```cpp
// 문자형 자료형
// ASCII Table에서 매칭되는 문자를 보여준다.
char c = 49;
wchar_t wc = 49;

// 둘은 다른 의미
char c1 = 1;    // 정수 1이 들어간다
c1 = '1';  // 문자 '1'에 해당하는 정수 49가 들어간다

// 문자열의 끝에는 NUL문자에 해당하는 0이 들어간다.
```
